Detailed Description

A table of hard-coded values for Landau's function \(g(n)\), the largest order of a permutation in the symmetric group \(S_n\), that is,

\[g(n) = \max_{\phi \in S_n} \min \{i \geq 1 \mid \phi^i = 1\},\]

where \(1\) denotes the identity permutation. This function is named for the mathematician Edmund Landau (1877-1938) pictured below.
